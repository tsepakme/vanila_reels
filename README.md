# Vertical Video Feed (Vanilla JS)

TikTok/Reels-like vertical video feed built using vanilla JavaScript.

## What's inside

This project does not use any third-party APIs; mock data is used to display Reels

### Main features:
- Clicking on a Reel or pressing the spacebar or 'K' key pauses it
- Clicking the sound icon or 'M' key toggles the sound on or off (it's muted by default)
- You can use the up and down arrows to navigate

## Run

Use any static server:

```npx serve .```

Open:

```http://localhost:3000```
